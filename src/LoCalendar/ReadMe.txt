System requirement:
- Minimum SDK : android API level 11 (os version 4.0 ice cream sandwich)
- Minimum screen resolution :  WVGA 480x800
- GPS service enabled

How to install to your phone
- open directory /LoCalendar/bin/
- copy LoCalendar.apk to your internal or external storage of your phone
- run file manager inside your phone and install the apk
- you will be propted to enabled permission to access the GPS and Calendar Provider
- run the application
- while running make sure your GPS or network provider is enable

Enjoy your application
